Guide d'utilisation

🚀 Premiers pas

  1. Lancez l'application MyBudget
  2. Choisissez la section Commun ou Perso
  3. Appuyez sur + pour ajouter votre première transaction
  4. Sélectionnez le type (Revenu ou Dépense) et la catégorie
  5. Entrez le montant et sauvegardez

👫 Budget Commun

La section Commun est conçue pour les dépenses partagées entre partenaires:

  • Coût total - Le montant total de la dépense
  • Pourcentage - Votre part de contribution (ex: 50%)
  • Contribution - Calculée automatiquement (Coût × %)

Le solde du budget Commun est automatiquement reporté dans votre section Perso comme dépense.

💵 Budget Personnel

La section Perso affiche:

  • Vos revenus personnels (Paie, Bonus, etc.)
  • Vos dépenses personnelles
  • Votre contribution au budget Commun (automatique)
  • Le solde disponible après toutes les dépenses

🔄 Transactions Récurrentes

Pour les dépenses mensuelles récurrentes (loyer, abonnements, etc.):

  1. Glissez une transaction vers la droite
  2. Ou utilisez le menu contextuel (appui long)
  3. Sélectionnez Dupliquer au mois suivant

La transaction sera copiée avec la même date au mois suivant.

📱 Multi-appareils

MyBudget synchronise automatiquement vos données via iCloud:

  • Assurez-vous d'être connecté au même compte iCloud sur tous vos appareils
  • Activez iCloud pour l'app dans Réglages > [Votre nom] > iCloud
  • Les modifications se synchronisent automatiquement toutes les 60 secondes
  • L'icône iCloud dans la barre de navigation indique le statut de synchronisation
  • Sur iOS: tirez vers le bas pour forcer la synchronisation
  • Sur macOS: utilisez Cmd+R pour synchroniser manuellement

📤 Export et Import

Gérez vos données avec les options d'export/import:

  • Export JSON - Sauvegardez toutes vos transactions en format JSON
  • Import JSON - Restaurez vos données avec détection automatique des doublons
  • Export Adhoc CSV - Exportez les transactions Adhoc en CSV
  • Import depuis courriel - Collez un courriel formaté pour créer une transaction

📊 Rapports

Analysez vos finances avec les rapports:

  • Filtrez par année, mois, section (Base/Perso) et récurrence
  • Visualisez les totaux par catégorie
  • Consultez le total général de la période

📧 Partage par courriel

Pour envoyer une transaction par courriel:

  1. Appui long sur une transaction (ou clic droit sur Mac)
  2. Sélectionnez "Envoyer par courriel"
  3. Le courriel sera pré-rempli avec les détails de la transaction

Format d'import par courriel:

DATE: 2026-03-15
SECTION: Base
TYPE: Dépense
CATEGORIE: Epicerie
MONTANT: 150.00
RECURRENT: Non
NOTE: Courses de la semaine

🛠️ Maintenance

Options de maintenance disponibles dans le menu Options > Maintenance:

  • Supprimer les doublons - Nettoie les transactions en double
  • Vider la BD - Supprime toutes les transactions (avec confirmation)

Questions fréquentes

Comment fonctionne le calcul du budget Commun dans Perso?

Dans la section Perso, les dépenses du budget Commun et le solde Commun sont automatiquement ajoutés comme dépenses. Cela représente votre contribution totale au budget partagé.

Puis-je entrer des montants négatifs?

Oui, les montants négatifs sont supportés pour les remboursements, crédits ou ajustements. Utilisez simplement le signe moins (-) devant le montant.

Mes données sont-elles sécurisées?

Oui, vos données sont stockées dans votre base de données iCloud privée. Elles sont chiffrées et accessibles uniquement par vos appareils connectés à votre compte Apple.

Puis-je partager mon budget avec mon partenaire?

Actuellement, chaque utilisateur a sa propre base de données. Pour un budget commun, vous pouvez chacun entrer vos contributions respectives avec votre pourcentage de participation.

Comment modifier une transaction?

Appuyez sur une transaction dans la liste pour ouvrir l'écran de modification. Vous pouvez aussi utiliser le menu contextuel (appui long) et sélectionner "Modifier".

Les données se synchronisent-elles entre iPhone et Mac?

Oui, tant que vous êtes connecté au même compte iCloud et que la synchronisation est activée, vos données sont disponibles sur iPhone, iPad et Mac.
