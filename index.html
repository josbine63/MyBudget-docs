Guide d'utilisation

📱 Les 5 onglets

L'application est organisee en 5 onglets principaux:

  • Sommaire - Vue d'ensemble du mois: debut, revenus, depenses, fin
  • Base - Budget partage (depenses communes avec pourcentage)
  • Perso - Vos revenus et depenses personnelles
  • Visa - Toutes les transactions payees par Visa
  • Marge - Gestion de votre marge de credit

📋 Sommaire

L'onglet Sommaire affiche une vue d'ensemble de votre mois:

  • Debut - Solde au debut du mois (calcule automatiquement)
  • Revenus - Total des revenus Perso du mois
  • Depenses - Compte a payer + Visa + Depenses Perso
  • Fin - Solde prevu a la fin du mois
  • Compte a payer - Montant a transferer pour le budget Base
  • Visa a payer - Total des achats Visa du mois
  • Marge a rembourser - Solde de la marge de credit

🏠 Budget Base

La section Base est concue pour les depenses partagees entre partenaires:

  • Cout total - Le montant total de la depense
  • Pourcentage - Votre part de contribution (ex: 50%)
  • Contribution - Calculee automatiquement (Cout x %)
  • Budget - Cochez pour marquer comme montant prevu (vs reel)

Le header affiche "Payable au compte" (budget - reel) et "Deductions" (depenses reelles).

💵 Budget Perso

La section Perso affiche:

  • Debut - Solde au debut du mois (editable pour le premier mois)
  • Revenus - Paie, bonus, allocations, etc.
  • Depenses - Vos depenses personnelles (non-Visa)

Le solde de fin est calcule: Debut + Revenus - (Compte a payer + Visa + Depenses).

💳 Onglet Visa

L'onglet Visa regroupe toutes les transactions payees par Visa:

  • Precedent - Solde Visa du mois precedent (editable)
  • Total du mois - Somme des achats Visa ce mois
  • Solde - Total a payer (precedent + mois)

Les transactions Visa peuvent provenir de Base ou Perso.

🏦 Marge de credit

Gerez votre marge de credit:

  • Limite - Votre limite de credit (editable)
  • Utilise - Montant actuellement emprunte
  • Disponible - Credit restant disponible
  • Emprunt - Ajouter un retrait de la marge
  • Remboursement - Enregistrer un paiement

🔄 Transactions Recurrentes

Pour les depenses mensuelles (loyer, abonnements, etc.):

  • Activez le toggle Recurrent lors de la creation
  • La transaction apparaitra automatiquement chaque mois
  • Sauter un mois: Desactivez le toggle Recurrent dans le mois a sauter
  • L'app termine la recurrence au mois precedent et en cree une nouvelle le mois suivant

📊 Suivi Budget

Comparez votre budget prevu vs depenses reelles:

  • Accessible via le bouton "Suivi" dans le menu
  • Affiche par categorie: budgete, depense, restant
  • Barre de progression coloree (vert/rouge)
  • Pourcentage d'utilisation du budget

📱 Multi-appareils

MyBudget synchronise automatiquement vos donnees via iCloud:

  • Connectez-vous au meme compte iCloud sur tous vos appareils
  • Activez iCloud pour l'app dans Reglages > [Votre nom] > iCloud
  • Sur iOS: tirez vers le bas pour forcer la synchronisation
  • Sur macOS: utilisez Cmd+R pour synchroniser manuellement

📤 Export et Import

Gerez vos donnees avec les options d'export/import:

  • Export JSON - Sauvegardez toutes vos transactions
  • Import JSON - Restaurez avec detection des doublons
  • Export Adhoc CSV - Exportez les transactions Adhoc
  • Import depuis courriel - Collez un courriel formate

📧 Partage par courriel

Pour envoyer une transaction par courriel:

  1. Appui long sur une transaction (ou clic droit sur Mac)
  2. Selectionnez "Envoyer par courriel"
  3. Le courriel sera pre-rempli avec les details

Format d'import par courriel:

DATE: 2026-03-15
SECTION: Base
TYPE: Depense
CATEGORIE: Epicerie
MONTANT: 150.00
RECURRENT: Non
NOTE: Courses de la semaine

👥 Partage Base avec partenaire

Synchronisez automatiquement vos transactions Base avec votre partenaire:

  • Activation - Activez "Partage Base" dans le menu Options
  • Sync automatique - Les transactions sont synchronisees toutes les 15 minutes
  • Sync manuelle - Utilisez le bouton "Sync Base maintenant" pour forcer une synchronisation
  • Pourcentage inverse - Le recepteur recoit automatiquement le pourcentage inverse (ex: si vous avez 60%, votre partenaire recoit 40%)
  • Budget et reel - Les transactions Budget et reelles sont toutes synchronisees

Les deux partenaires doivent activer "Partage Base" et etre connectes au meme compte iCloud.

Questions frequentes

Comment fonctionne le "Compte a payer" dans le Sommaire?

Le Compte a payer = Budget Base - Depenses reelles Base + Adhoc. C'est le montant net que vous devez transferer vers le compte commun.

Comment sauter un mois pour une transaction recurrente?

Allez dans le mois a sauter, ouvrez la transaction recurrente et desactivez le toggle "Recurrent". L'app terminera la recurrence au mois precedent et en creera une nouvelle a partir du mois suivant.

Quelle est la difference entre Budget et non-Budget dans Base?

Les transactions "Budget" representent vos previsions (montants planifies). Les transactions non-Budget sont les depenses reelles. Le suivi budget compare les deux.

Puis-je entrer des montants negatifs?

Oui, les montants negatifs sont supportes pour les remboursements, credits ou ajustements.

Comment modifier le solde de debut de mois?

Dans l'onglet Perso, appuyez sur le montant "Debut" pour le modifier. Ceci est utile pour le premier mois d'utilisation.

Les donnees se synchronisent-elles entre iPhone et Mac?

Oui, via iCloud. Assurez-vous d'etre connecte au meme compte iCloud et que la synchronisation est activee.

Comment fonctionne la marge de credit?

Definissez votre limite et solde initial. Les emprunts augmentent le solde utilise, les remboursements le reduisent. Le disponible = Limite - Utilise.

Comment partager mes transactions Base avec mon partenaire?

Activez "Partage Base" dans le menu Options. Les deux partenaires doivent activer cette option et etre connectes au meme compte iCloud. Les transactions sont synchronisees automatiquement toutes les 15 minutes, et le pourcentage est automatiquement inverse (100% - votre %).
